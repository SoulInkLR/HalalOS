# Pour compiler et lancer le halalOS, taper make qemu
# Pour connaître les commandes disponibles, tapez help
# Vous avez 4 shell à votre disposition (1, 2, 3, 4)
   => tappez "shell" suivi d'un espace et du numero du shell voulu
# Pour lancer une commande sans argument, tapez simplement son nom
# Si la commande a un paramètre, tapez le nom de la commande suivi d'un
  espace et de l'argument
  ex : test 4
